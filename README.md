# DollShelf Club MVP

Клікабельний MVP вебсоцмережі для колекціонерів ляльок.

## Як відкрити

Відкрийте `index.html` у браузері або запустіть локальний сервер з цієї папки:

```bash
python3 -m http.server 4173
```

Після цього відкрийте `http://localhost:4173`.

## Що є в MVP

- реєстрація та демо-вхід;
- створення і редагування профілю;
- профіль іншого користувача;
- стрічка постів;
- створення посту з фото, текстом і тегами;
- лайки та коментарі;
- підписка і відписка;
- пошук за нікнеймом, країною, інтересами та естетикою;
- мінімальна адмін-сторінка;
- локальне збереження даних у браузері.

Для швидкого тесту увійдіть як `NikaShelf`.
